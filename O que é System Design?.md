System design é o campo que debate diversas áreas de um sistema, como Arquitetura, Componentes, Módulos, Interfaces e fluxos desses sistema.

Tudo começa a partir de requisitos de negócio, uma aplicação precisa atender uma quantidade X de usuários, simultaneamente ou não, com Y % de disponibilidade e custo Z. O plano que viabiliza tudo isso tecnicamente é o System
